# SmartShell TV Display

Эта папка содержит фронтенд TV-экрана и вспомогательные Windows-скрипты.

## Что здесь находится

- `smartshell-display/` — HTML/CSS/JS экрана
- `tools/` — старые скрипты локального запуска и автозапуска под Windows

## Актуальный запуск

Сейчас основной запуск идёт не через `tools/run-display.ps1`, а через корневой `timer_server.py`.

После запуска сервера экран открывается по адресу:

```text
http://127.0.0.1:8080/tv/?tv=1&fullscreen=1
```
