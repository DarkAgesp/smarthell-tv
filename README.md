# SmartShell Display

Информационный дисплей для игрового клуба с интеграцией SmartShell API.

![Preview](preview.png)

## Возможности

- 📺 **TV-режим** - оптимизирован для 24/7 отображения на телевизоре
- 🎨 **Кастомные обои** - настройте фон под свой клуб
- 🖱️ **Drag & Drop** - перетаскивайте ПК между зонами
- 🔄 **Автообновление** - данные обновляются автоматически
- 📊 **Статусы ПК**:
  - 🟢 Свободен - готов к игре
  - 🔵 В работе - показывает время до конца сессии
  - 🟡 Забронирован - показывает время брони

## Быстрый старт

### 1. Получите API токен

1. Войдите в [панель администратора SmartShell](https://admin.smartshell.gg)
2. Перейдите в настройки -> API
3. Создайте новый токен доступа

### 2. Запустите дисплей

```bash
# Откройте index.html в браузере
# Или используйте локальный сервер:
npx serve .
```

### 3. Настройте подключение

1. Нажмите кнопку ⚙️ в правом нижнем углу
2. Введите ваш API токен
3. Нажмите "Сохранить"

## Настройки

### API Endpoint
```
https://api.smartshell.gg/graphql
```

### Интервал обновления
По умолчанию: 30 секунд. Можно изменить в настройках (10-300 сек).

### Кастомные обои
Поддерживаются:
- Градиенты (по умолчанию)
- Изображения по URL
- Сплошной цвет

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `Ctrl+T` | Включить/выключить TV режим |
| `Ctrl+D` | Включить/выключить режим перетаскивания |
| `Esc` | Закрыть панель настроек |

## API Endpoints

Дисплей использует следующие GraphQL запросы:

- `hostsOverview` - получение информации о всех ПК
- `getBookings` - получение активных бронирований
- `myClub` - информация о клубе

## Демо-режим

Если API токен не настроен, дисплей работает в демо-режиме с тестовыми данными.
